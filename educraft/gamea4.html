How to Play

Movement

W / SForward / backward (Backwards is broken)
A / DStrafe left / right
SpaceJump
ShiftSprint
ESCPause
EInventory
ICreative Menu (all blocks)
GSpawn test mob
Ctrl+EExport world

Combat

Left clickBreak block / Punch / Attack
Right clickPlace block / Use item

Swords and Axes deal extra damage. Armour reduces damage received. Eat food (Bread, Generic Food) to restore health. You have 10 hearts (20 HP).

Farming

Hoe on Dirt → Farmland. Seeds on Farmland → Wheat (3 stages). Break grown wheat to harvest.

Multiplayer (LAN)

One player hosts, others join using the Host's room code. Set a username and colour before connecting. You gotta be on the same Internet connection!
